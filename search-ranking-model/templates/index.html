Model architecture

The model fuses four independent signals using a normalized weighted sum.

score = Σ(wᵢ × signalᵢ) / Σwᵢ
= w₀·tfidf + w₁·ctr·eng + w₂·authority + w₃·recency
≈
TF-IDF
Term frequency × inverse document frequency with a 1.8× title boost. Measures how relevant the document's vocabulary is to the query.
tf(t,d) × log(N/df(t)) × boost
↑
CTR × Engagement
Click-through rate normalized to [0,1], multiplied by an engagement quality multiplier derived from average session time and inverse bounce rate.
ctr × ((t/300) + (1-bounce)) / 2
★
Authority
A domain trust score between 0 and 1 representing backlink quality, domain age, and trust signals. Acts as a prior on document quality.
authority ∈ [0.0, 1.0]
⏱
Recency
Linear decay from 1.0 (fresh) to 0.0 (600+ days old). Recent documents score higher when the recency weight is elevated.
max(0, 1 - days / 600)
⚖
Weight normalization
Weights are always normalized to sum to 1.0 before scoring, so you can set raw values freely without worrying about scale.
wᵢ_norm = wᵢ / Σwⱼ
∅
Zero dependencies
Built with Python's standard library only — no NumPy, no scikit-learn. The IDF cache avoids recomputation across queries on the same corpus.
import math, json # that's it

Core ranking logic

The complete scoring pipeline — readable, testable, extendable.

src/ranking_model.py
def score_document(self, doc: Document, query: str) -> ScoredResult:
    w = self.weights.normalized()

    # Signal 1: lexical TF-IDF with title boost
    tfidf_s  = self._score_tfidf(doc, query)

    # Signal 2: CTR scaled by engagement quality
    ctr_s    = self._score_ctr(doc)   # ctr × engagement

    # Signal 3: domain authority prior
    auth_s   = self._score_authority(doc)

    # Signal 4: linear time decay
    rec_s    = self._score_recency(doc)

    final = (
        w.tfidf     * tfidf_s +
        w.ctr       * ctr_s   +
        w.authority * auth_s  +
        w.recency   * rec_s
    )
    return ScoredResult(doc, final, tfidf_s, ctr_s, auth_s, rec_s)

def search(self, query: str, top_k: int = 10) -> List[ScoredResult]:
    """Return top-k ranked results for a query."""
    scored = [self.score_document(doc, query) for doc in self.corpus]
    scored.sort(key=lambda r: r.final_score, reverse=True)
    for i, result in enumerate(scored):
        result.rank = i + 1
    return scored[:top_k]
